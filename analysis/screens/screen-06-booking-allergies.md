# Бронирование: аллергии

## Назначение
Позволить клиенту указать аллергии для текущей брони и сохранить их в профиле.

## Пользовательские роли
- Клиент.

## Источники требований
- Domain: Allergen, 5.3, BR-06, BR-12; BR-10; FR-13, FR-14; US-10, US-11; UC-04; NFR-07, NFR-11.

## Когда отображается
- После шага выбора проката.

## Откуда можно перейти
- «Бронирование: прокат экипировки».

## Куда можно перейти
- «Бронирование: итоговая сводка».
- Назад к прокату.

## Основной пользовательский сценарий
1. Система показывает справочник аллергенов.
2. Клиент выбирает несколько аллергенов или оставляет выбор пустым.
3. Система использует сохранённые аллергии клиента при следующих бронированиях.
4. Клиент переходит к сводке.

## Состав экрана
- Список аллергенов: глютен, молочные продукты, яйца, рыба, ракообразные, моллюски, арахис, орехи, соя, кунжут, сельдерей, горчица, люпин, сульфиты.
- Множественный выбор.
- Подсказка, что аллергии необязательны и сохраняются в профиле. **Inferred**.
- CTA «Далее».

## Действия пользователя
- Выбрать/снять аллергены.
- Продолжить без аллергий.
- Вернуться назад.

## Связанные Bottom Sheets
- Нет.

## Состояния
- Default: список аллергенов, возможно с предвыбранными сохранёнными аллергиями.
- Loading: загрузка справочника/сохранённых аллергий.
- Empty: справочник не вернул аллергены. **Missing**.
- Error: «Ошибка сервера».
- Success: выбор добавлен в сводку.
- Disabled: не требуется; шаг можно пройти без выбора. **Inferred**.
- Offline: не описано. **Missing**.

## Валидация
- Допускается множественный выбор из справочника.
- Свободный ввод аллергий не описан и не должен добавляться. **Missing/Inferred constraint**.

## Ограничения
- Аллергии являются чувствительными данными и доступны только авторизованному клиенту.

## Edge Cases
- Сохранённые аллергии отсутствуют.
- Справочник аллергенов недоступен.

## Inferred
- При повторной брони сохранённые аллергии должны быть предвыбраны, так как US-11 требует использовать сохранённые аллергии.

## Missing
- Можно ли добавить «другое», можно ли снять сохранённые аллергии для будущего профиля, точный механизм сохранения.

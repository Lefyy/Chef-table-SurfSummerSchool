# Chef Table API specification

Redocly/OpenAPI 3.1 YAML specification for the client-facing MVP API.

## Entry point

- `openapi.yaml`

## Scope

The specification is based only on `analysis/screens/`, `analysis/5-design/api-sequence.md`, and `analysis/5-design/data-model.md`.
It covers SMS authentication, schedule browsing, slot details, booking setup dictionaries, booking creation and cancellation, profile, saved allergies, history/details, chef rating, and logout.

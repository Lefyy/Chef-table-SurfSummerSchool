# Feature: Бронирование кулинарного класса

## Описание

Клиент может выбрать доступный класс из расписания, посмотреть детали и создать бронь на одно место. При бронировании можно выбрать доступный прокат и указать аллергены.

## Основной сценарий

1. Клиент открывает расписание.
2. Переходит в детали выбранного слота.
3. Открывает форму бронирования.
4. Выбирает прокат и аллергены при необходимости.
5. Подтверждает бронь.
6. Приложение создаёт бронь и показывает экран успешного бронирования.

## Правила и ограничения

- Бронь создаётся только при наличии свободных мест.
- На одну бронь резервируется одно место.
- Отменённый студией класс нельзя забронировать повторно.
- Недоступный прокат не должен быть выбран клиентом.
- Выбранные аллергии сохраняются в профиле клиента.
